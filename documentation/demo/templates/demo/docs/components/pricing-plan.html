Pricing Plan

A flexible pricing plan component for displaying individual pricing tiers with features, pricing, and call-to-action buttons. 

<c-pricing_plan 
    title="Starter" 
    description="Perfect for getting started"
    price="€19" 
    period="/month" 
    :features="['5 projects', 'Email support', '10GB storage']"
    button_label="Get Started"
    button_variant="primary"
></c-pricing_plan>

Component API

Property Type Default Description
title string - Plan title/name (required)
description string - Short description of the plan
price string - Plan price (required)
period string - Billing period (e.g., "/month", "/year")
discount_price string - Original price (shows as strikethrough)
features list - List of features (use :features for arrays)
button_label string - Call-to-action button text
button_variant string primary Button style variant
button_size string md Button size
highlight string false Highlight the plan with border (use "true")
badge string - Badge text (e.g., "Popular", "Best Value")

Examples

Basic Plans

Simple pricing plans with essential features. 

<c-pricing_plan 
    title="Basic" 
    price="€9" 
    period="/month" 
    :features="['1 project', 'Email support', '1GB storage']"
    button_label="Start Free"
    button_variant="outline"
></c-pricing_plan>

Highlighted Plan

Use the highlight and badge props to make a plan stand out. 

<c-pricing_plan 
    title="Pro" 
    description="Most popular choice"
    price="€29" 
    period="/month" 
    :features="['10 projects', 'Priority support', '10GB storage']"
    button_label="Choose Pro"
    button_variant="primary"
    highlight="true"
    badge="Popular"
></c-pricing_plan>

Discount Pricing

Show original pricing with discounts using the discount_price prop. 

<c-pricing_plan 
    title="Annual Pro" 
    description="Save 20% with yearly billing"
    price="€297" 
    discount_price="€348"
    period="/year" 
    :features="['10 projects', 'Priority support', '10GB storage']"
    button_label="Save 20%"
    button_variant="primary"
    badge="Best Value"
></c-pricing_plan>

Custom Content

Use the default slot to add custom content at the bottom of the plan.
