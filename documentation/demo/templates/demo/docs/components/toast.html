Appendix

Implementation Guidelines

Base Template Setup

Include <c-toast-area></c-toast-area> in your base template, typically near the opening <body> tag.

Server-Side Implementation

Toasts are typically triggered by server responses. Implement appropriate Django views to return toast notifications with HTMX.

Auto-Dismiss

Toasts should automatically dismiss after a reasonable time (3-5 seconds) to avoid cluttering the interface.

Message Guidelines

Keep toast messages concise and actionable. Use clear, user-friendly language and avoid technical jargon.

Django View Example

Server-Side Toast Generation

Exemple de vue Django utilisant les classes Toast pour générer des notifications avec HTMX.
# views.py
from django.shortcuts import render
from django.http import HttpResponse
from django_components_ui import Toast, ToastSuccess, ToastError, ToastWarning, ToastInfo
import time

def toast_action(request):
    """Vue de démonstration pour tous les types de toasts."""
    toast_type = request.GET.get('type', 'success')

    # Simulation d'une opération qui prend du temps
    time.sleep(1)

    # Utilisation des classes Toast spécialisées
    if toast_type == 'success':
        return ToastSuccess("Opération réussie avec succès !")
    elif toast_type == 'error':
        return ToastError("Une erreur s'est produite")
    elif toast_type == 'warning':
        return ToastWarning("Attention, vérifiez vos données")
    elif toast_type == 'info':
        return ToastInfo("Information importante")
    else:
        # Utilisation avec paramètres personnalisés
        return Toast(
            content="Toast personnalisé", 
            variant=toast_type,
            duration=5000,  # Durée personnalisée en millisecondes
            dismissible=True  # Possibilité de fermer manuellement
        )

Classes Toast Disponibles

  • ToastSuccess(message) - Toast de succès
  • ToastError(message) - Toast d'erreur
  • ToastWarning(message) - Toast d'avertissement
  • ToastInfo(message) - Toast d'information
  • Toast(content, variant, duration, dismissible) - Toast personnalisé
