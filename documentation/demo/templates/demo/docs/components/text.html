Text

A versatile text component with support for different variants and custom colors.
Default text content 
<c-text>Default text content</c-text>

Component API

Property Type Default Description
variant string default Text style: default, strong, subtle
color string - Tailwind color class (e.g., "red-500", "blue-100")
class string - Additional CSS classes

Examples

Text Variants

The text component supports three style variants via the variant prop.
This is default text with normal weight and standard color
This is strong text with semi-bold weight and darker color
This is subtle text with normal weight and lighter color
<c-text variant="default">This is default text</c-text>
<c-text variant="strong">This is strong text</c-text>
<c-text variant="subtle">This is subtle text</c-text>

Custom Colors

Override the default color with any Tailwind color class using the color prop.
This is red text
This is blue text
This is green text
This is purple text
This is orange text
<c-text color="red-500">This is red text</c-text>
<c-text color="blue-600">This is blue text</c-text>
<c-text color="green-500">This is green text</c-text>
<c-text color="purple-500">This is purple text</c-text>
<c-text color="orange-500">This is orange text</c-text>

Variants with Custom Colors

Combine variants with custom colors for more styling options.
Strong red text
Subtle blue text
Default green text
<c-text variant="strong" color="red-600">Strong red text</c-text>
<c-text variant="subtle" color="blue-400">Subtle blue text</c-text>
<c-text variant="default" color="green-600">Default green text</c-text>

Additional Styling

Use the class prop to add extra styling with Tailwind classes.
Large text
Small text
Uppercase with letter spacing
Italic text
Underlined text
Centered text (with block display)
<c-text class="text-lg">Large text</c-text>
<c-text class="text-sm">Small text</c-text>
<c-text class="uppercase tracking-wide">Uppercase with letter spacing</c-text>
<c-text class="italic">Italic text</c-text>
<c-text class="underline">Underlined text</c-text>
<c-text class="text-center block">Centered text</c-text>

Real-world Usage

Examples of how the text component can be used in typical UI patterns.
Product Title This is a detailed description of the product with all the important information customers need to know. Last updated: January 2025
✓ Success: Operation completed successfully ⚠ Warning: Please review your settings ✗ Error: Something went wrong 
<!-- Card content -->
<c-text variant="strong" class="text-xl mb-2 block">Product Title</c-text>
<c-text variant="default" class="mb-3 block">Product description...</c-text>
<c-text variant="subtle" class="text-sm block">Last updated: January 2025</c-text>

<!-- Status messages -->
<c-text variant="strong" color="green-600">✓ Success message</c-text>
<c-text variant="default" color="yellow-600">⚠ Warning message</c-text>
<c-text variant="subtle" color="red-500">✗ Error message</c-text>

<!-- Form helper text -->
<c-text variant="strong" class="mb-1 block">Field Label</c-text>
<c-text variant="subtle" class="text-sm mt-1 block">Helper text</c-text>

Appendix

Design Guidelines

Semantic Usage

Use strong for important text, subtle for secondary information, and default for regular content.

Display Types

By default, text renders as span (inline). Use class="block" for block-level display.
