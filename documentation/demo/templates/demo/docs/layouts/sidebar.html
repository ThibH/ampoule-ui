Sidebar

A collapsible sidebar navigation component for organizing application navigation and content.

Navigation

  • Dashboard
  • Projects
  • Settings
  • Help

Main Content

This is the main content area that sits beside the sidebar.

 
<c-layout.sidebar>
    <c-slot name="content">
        <div class="menu p-4 w-80 min-h-full bg-base-200">
            <h3 class="text-lg font-bold mb-4">Navigation</h3>
            <ul>
                <li><a href="#" class="active">Dashboard</a></li>
                <li><a href="#">Projects</a></li>
                <li><a href="#">Settings</a></li>
                <li><a href="#">Help</a></li>
            </ul>
        </div>
    </c-slot>
    <c-slot name="main">
        <div class="p-8">
            <h2 class="text-2xl font-bold mb-4">Main Content</h2>
            <p>This is the main content area.</p>
        </div>
    </c-slot>
</c-layout.sidebar>

Component API

Slot Required Description
content Yes Sidebar navigation content
main Yes Main content area beside the sidebar

Appendix

Mobile Considerations

Responsive Design

On mobile devices, consider using a drawer-based sidebar that can be toggled rather than always visible.

Width Control

Use fixed widths like w-64 or w-80 for consistent sidebar sizing across different screen sizes.
