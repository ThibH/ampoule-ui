Navbar

A responsive navigation bar component with support for logos, navigation links, and action buttons.

MyApp

 Home About Login 
<c-layout.navbar>
    <c-slot name="logo">
        <h1 class="text-xl font-bold">MyApp</h1>
    </c-slot>
    <c-slot name="nav">
        <c-navbar.item href="#" icon="home">Home</c-navbar.item>
        <c-navbar.item href="#" icon="info">About</c-navbar.item>
        <c-navbar.item href="#" icon="user">Login</c-navbar.item>
    </c-slot>
</c-layout.navbar>

Component API

Slot Required Description
logo No Left side logo or brand content
nav No Right side navigation links and actions

With Dropdown Menu

Navbar with a dropdown menu for additional navigation options.

Dashboard Home Profile Settings Logout 
<c-layout.navbar>
    <c-slot name="logo">
        <span class="text-xl font-bold">Dashboard</span>
    </c-slot>
    <c-slot name="nav">
        <c-navbar.item href="#" icon="home">Home</c-navbar.item>
        <c-navbar.submenu label="Account" icon="user">
            <c-navbar.item href="#" icon="user">Profile</c-navbar.item>
            <c-navbar.item href="#" icon="settings">Settings</c-navbar.item>
            <c-navbar.item href="#" icon="log-out">Logout</c-navbar.item>
        </c-navbar.submenu>
    </c-slot>
</c-layout.navbar>

Responsive Behavior

Mobile Optimization

Fixed Positioning
