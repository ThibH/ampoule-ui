Footer

A flexible footer component for organizing links, contact information, and legal content at the bottom of pages.

  • Careers
  • Contact

  • Terms of Service

<c-layout.footer>
    <c-slot name="content">
        <div class="grid grid-cols-1 md:grid-cols-3 gap-8">
            <div>
                <h3 class="text-lg font-bold mb-4">Company</h3>
                <ul class="space-y-2">
                    <li><a href="#" class="link link-hover">About Us</a></li>
                    <li><a href="#" class="link link-hover">Careers</a></li>
                    <li><a href="#" class="link link-hover">Contact</a></li>
                </ul>
            </div>
        </div>
    </c-slot>
</c-layout.footer>

Component API

Slot Required Description
content Yes Footer content including links, text, and other elements

Appendix

Sticky Footer

Page Layout

To make the footer stick to the bottom of the page, use flexbox layout on your page container with min-h-screen flex flex-col.

Responsive Design

Consider stacking footer columns on mobile devices using responsive grid classes like grid-cols-1 md:grid-cols-3.
