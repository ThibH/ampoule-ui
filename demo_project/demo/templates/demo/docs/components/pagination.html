Pagination

A flexible pagination component that adapts to different result set sizes with smart page numbering and navigation controls. 

<c-pagination current_page="3" total_pages="10" base_url="{% url 'your_view' %}" />

Component API

Property Type Default Description
current_page number 1 The currently active page number
total_pages number 1 Total number of pages available
base_url string # Base URL for pagination links (page parameter will be appended)
simple boolean false Use simple pagination (Previous/Next only)
size string md Button size: sm, md, lg
class string - Additional CSS classes

Appendix

Django Integration

Django Paginator Integration

Use with Django's built-in Paginator for seamless pagination in your views.

Django View Example:

from django.core.paginator import Paginator

def item_list(request):
    items = Item.objects.all()
    paginator = Paginator(items, 10)  # 10 items per page
    page_number = request.GET.get('page', 1)
    page_obj = paginator.get_page(page_number)
    
    return render(request, 'items/list.html', {
        'page_obj': page_obj,
    })

Template Usage:

<!-- Display items -->
{% for item in page_obj %}
    <div>{{ item.name }}</div>
{% endfor %}

<!-- Pagination component -->
<c-pagination 
    current_page="{{ page_obj.number }}" 
    total_pages="{{ page_obj.paginator.num_pages }}" 
    base_url="{% url 'items:list' %}" />

URL Parameters

The component automatically appends ?page=N to the base URL. Make sure your view handles the page parameter.

HTMX Integration

HTMX Compatible

Pagination links work seamlessly with HTMX for dynamic content loading without page refreshes.

HTMX Template Example:

<div id="content" hx-target="#content" hx-swap="outerHTML">
    <!-- Your paginated content -->
    {% for item in page_obj %}
        <div>{{ item.name }}</div>
    {% endfor %}
    
    <!-- Add hx-get to pagination links -->
    <c-pagination 
        current_page="{{ page_obj.number }}" 
        total_pages="{{ page_obj.paginator.num_pages }}" 
        base_url="{% url 'items:list' %}" 
        hx-get="{% url 'items:list' %}" />
</div>
