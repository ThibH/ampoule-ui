Range

A range slider input component for selecting numeric values within a specified range. 

<c-range min="0" max="100" value="50" />

Component API

Property Type Default Description
min number 0 Minimum value of the range
max number 100 Maximum value of the range
value number 50 Current value of the range slider
step number 1 Step increment for the range values
name string - Form field name
id string - Unique identifier for the range input
disabled boolean false Disable the range input
class string - Additional CSS classes for the range input

Appendix

Usage Guidelines

Value Display

Consider adding a value display or input field alongside the range for precise control.

Step Size

Choose appropriate step sizes based on the expected precision and range of values.

Accessibility

Always provide descriptive labels and consider keyboard navigation for range inputs.
