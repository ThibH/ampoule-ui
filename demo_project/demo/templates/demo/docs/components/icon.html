Icon

A flexible icon component supporting various icon libraries with consistent sizing and styling. 

<c-icon name="home" class="size-6"></c-icon>

Component API

Property Type Default Description
name string required Icon name from the supported icon library
class string - Additional CSS classes (size-4, size-6, text-primary, etc.)
size string - Quick size preset (xs, sm, md, lg, xl)

    Appendix

    Usage Guidelines

    Icon Library

    Icons are typically provided by libraries like Lucide, Heroicons, or Feather. Check the available icon names in your chosen library.

    Consistent Sizing

    Use consistent icon sizes throughout your application. Common sizes are 16px (size-4), 20px (size-5), and 24px (size-6).

    Accessibility

    For decorative icons, ensure they don't interfere with screen readers. For functional icons, provide appropriate aria-labels.

    Performance

    Icon libraries are typically loaded once and cached. Choose a consistent library to avoid loading multiple icon sets.

    Icon Categories

    Navigation

    home, menu, arrow-left, arrow-right, chevron-up, chevron-down

    Actions

    Status
