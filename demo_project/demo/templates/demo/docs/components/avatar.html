Avatar

A flexible avatar component for displaying profile pictures, initials, or placeholders with various sizes and styles.

JD AB 
<!-- With image -->
<c-avatar src="image.jpg" size="w-12" alt="Profile picture"></c-avatar>

<!-- Placeholder with initials -->
<c-avatar placeholder size="w-12" bg_color="bg-primary" text_color="text-primary-content">JD</c-avatar>

Component API

Property Type Default Description
src string - Image URL for the avatar. If not provided, shows placeholder content
size string w-12 Size using Tailwind width classes (w-6, w-8, w-10, w-12, w-16, w-20, w-24, w-32)
shape string rounded-full Shape using Tailwind border-radius classes (rounded, rounded-xl, rounded-full)
online string - Online status indicator ("true" for online, "false" for offline)
placeholder boolean false Whether this is a placeholder avatar (for initials/text)
ring boolean false Whether to show a colored ring around the avatar
bg_color string bg-neutral Background color for placeholder avatars (Tailwind bg- classes)
text_color string text-neutral-content Text color for placeholder avatars (Tailwind text- classes)
alt string Avatar Alt text for image avatars (accessibility)
class string - Additional CSS classes for the avatar container

Large Team Example

JS AB CD +5 
<!-- Default spacing (3) -->
<c-avatar-group>
    <c-avatar placeholder size="w-10" bg_color="bg-primary" text_color="text-primary-content">A</c-avatar>

Avatar Group API

The c-avatar-group component provides a convenient wrapper for displaying multiple avatars in overlapping groups.

Property Type Default Description
spacing string 3 Numeric spacing value (1=tight, 6=wide). Lower numbers create tighter overlapping, higher numbers create wider spacing
class string - Additional CSS classes for the avatar group container

Appendix

Usage Guidelines

Accessibility

Always provide meaningful alt text for image avatars and ensure sufficient color contrast for placeholder avatars.

Image Loading

Consider providing fallback initials or placeholders for cases where profile images fail to load.

Consistency

Use consistent sizing and colors throughout your application for a cohesive user experience.
