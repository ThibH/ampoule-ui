Dropdown

A versatile dropdown menu component with support for various placements, triggers, and structured menu content using the new menu components.

Home About Contact 
<c-dropdown label="Menu">
    <c-menu.item icon="home">Home</c-menu.item>
    <c-menu.item icon="info">About</c-menu.item>
    <c-menu.item icon="mail">Contact</c-menu.item>
</c-dropdown>

Component API

Property Type Default Description
label string required Text displayed on the dropdown trigger button
icon string - Icon name to display in the dropdown button
variant string neutral Button style: primary, secondary, accent, neutral, outline, ghost
placement string bottom Menu position: top, bottom, start, end, center
hover boolean false Open dropdown on hover instead of click
class string - Additional CSS classes for the dropdown container

Appendix

Usage Guidelines

Menu Structure

Use <c-menu.item> components directly within the dropdown for menu items.

Placement Considerations

Choose appropriate placement based on dropdown position to avoid viewport overflow.

HTMX Integration

Dropdown buttons and menu items work seamlessly with HTMX attributes for dynamic actions.
