Switch

A toggle switch component for binary choices with support for labels, descriptions, and flexible orientation. Built on DaisyUI toggle component. 

<c-form.switch label="Enable notifications" />

Component API

Property Type Default Description
field Django Field - Django form field for automatic integration
label string - Label text displayed next to the switch
description string - Help text displayed below the label
orientation string horizontal Layout orientation: horizontal, vertical
align string right Switch alignment: left, right (only for horizontal orientation)
size string md Switch size: xs, sm, md, lg
variant string default Color variant: default, primary, secondary, accent, info, success, warning, error
disabled boolean false Disable the switch
checked boolean false Initial checked state
id string - Unique identifier for the switch
name string - Form field name
value string - Value when checked

Orientation and Alignment

Control the layout orientation and switch position relative to the label.

Horizontal - Right aligned (default)

Horizontal - Left aligned

Vertical

<!-- Horizontal layout - right aligned (default) -->
<c-form.switch 
    label="Right aligned switch" 
    description="Switch appears after the label text" />

<!-- Horizontal layout - left aligned -->
<c-form.switch 
    align="left"
    label="Left aligned switch" 
    description="Switch appears before the label text" />

<!-- Vertical layout -->
<c-form.switch 
    orientation="vertical" 
    label="Push notifications" 
    description="Receive push notifications on your device" />

Django Form Integration

Using with Django Forms

The component automatically integrates with Django form fields when you pass the field parameter. Works best with BooleanField.

# In your Django form:
class SettingsForm(forms.Form):
    notifications = forms.BooleanField(required=False)
    dark_mode = forms.BooleanField(required=False)

# In your template:
<c-form.switch field="{{ form.notifications }}" label="Enable Notifications" />
<c-form.switch field="{{ form.dark_mode }}" label="Dark Mode" variant="accent" /> 
<!-- Django form integration -->
<c-form.switch field="{{ form.notifications }}" label="Enable Notifications" />
<c-form.switch field="{{ form.dark_mode }}" label="Dark Mode" variant="accent" />

<!-- The component will automatically: -->
<!-- - Set name, id, value from the field -->
<!-- - Handle checked state from field value -->
<!-- - Display validation errors if any -->

BooleanField Integration

The switch component works seamlessly with Django's BooleanField, automatically handling the checked state and form submission.

Error Handling

Form validation errors are automatically displayed below the switch with proper styling.

Accessibility

Always provide proper labels for better accessibility. The component automatically links labels to switches for proper form semantics.
